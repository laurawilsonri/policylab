# policylab
Wagtail site for testing model translation on policylab

# Install:
git clone <clone_url>
cd policylab
python3 -m venv plab/env
source plab/env/bin/activate
pip install wagtail

# How to run:
cd policylab
source plab/env/bin/activate
python manage.py runserver

Then, navigate to http://127.0.0.1:8000/admin to see CMS!
